Inserts the given [[String]] argument at offset/index given by argument of type [[Integer]] 'self', [[Return value]] is 'self':
```ruby
'foo'.insert(1, 'bar') # => "fbaroo"
'foo'.insert(-2, 'bar') # => "fobaro"
```
