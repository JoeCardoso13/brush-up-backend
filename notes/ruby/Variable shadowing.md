When defining the parameter of a [[Block]], hence having [[Layered Scope]] rules, it is possible to avoid access to outer scope [[Local variable]] by naming the parameter the same as the local variable in the outer scope. Thence said variable would not be affected by the execution of the [[Block]]. 

It mostly happens accidentally.
