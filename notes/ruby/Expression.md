_Ruby expressions always return a value_, even if that value is `nil` or an [[Exception]]. For clarity, we call the [[Return value]] of an ordinary Ruby *expression* as **what it evaluates to**. An *expression* is anything that can be evaluated, and pretty much everything you write in Ruby is an expression.  

"*In Ruby, everything is an expression.*"
