Accepts [[Character selector]]s.

There is a [[Destructive method]] version of this [[Method]] with `!` appended.

Each character specified by string the [[Character selector]] is translated to the corresponding character from second argument.

```ruby
'hello'.tr('el', 'ip') #=> "hippo"
```
