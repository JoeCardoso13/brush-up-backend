Are typically used as identifiers, when the intent is to have unique, [[Immutable]] notation, unlike [[String]]. They are created with semicolon before a word ([[Literal]]). 
* If defined as a [[String]], they don't need to be valid variable name, e.g. can contain spaces.
* [[Immutable]]. 
* Conventionally used as keys in [[Hash]].
