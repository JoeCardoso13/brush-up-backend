A block is a related set of JavaScript [[Statement]]s and [[Expression]]s between a pair of opening and closing curly braces.

Not everything between curly braces is technically a block. For instance, the braces that surround an [[Object]]s in [[Literal form]] do **not** define a block. Technically, the braces that surround a [[Function]] body don't define a block either, though it is convenient to think of [[Function]] bodies as blocks.

In general, blocks appear in [[if Statement]]s, [[While loop]]s, [[Do while loop]]s, [[For loop]]s, `switch`, and `try...catch` [[Statement]]s, or (less commonly) by themselves.
