Is the way to represent [[Value]]s from the different data [[Type]]s when writing them out. Examples:
```javascript
'Hello, world!'     // string literal
3.141592            // numeric literal
true                // boolean literal
{ a: 1, b: 2 }      // object literal
[ 1, 2, 3 ]         // array literal
undefined           // undefined literal
```
