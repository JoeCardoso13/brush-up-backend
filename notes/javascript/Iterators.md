The basic iterators are:
* [[While loop]]
* [[Do while loop]]
* [[For loop]]
But other common, [[Object]] defined [[Method]]s are:
* [[forEach]]
* [[For in]]
* [[Map]]
* [[Filter]]

There are keywords that help controlling the loop iterations (don't work with [[forEach]]!!!):
* `continue`: skip the current iteration and continue to the next one.
* `break`: breaks out of a loop and doesn't run any more iterations.
