Basic syntax:
```js
do {
  //block
} while (condition)
```
It runs the [[Block]] over and over again until the condition [[Expression]] evaluates to falsy. It checks the condition after running the [[Block]], so if condition is falsy the [[Block]] **runs** at least once.
